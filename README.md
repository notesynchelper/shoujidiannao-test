# shoujidiannao-test

obsync 客户端**测试通道**发行包。与生产 [`shoujidiannao`](https://github.com/notesynchelper/shoujidiannao) 同 `id`，仅用于内部测试。

## 安装

下载最新 release 里的 `main.js` / `manifest.json` / `styles.css`，放到 vault 的 `.obsidian/plugins/shoujidiannao/` 目录，重启 Obsidian 并启用。

> ⚠️ 与生产 `shoujidiannao` 共享同一 plugin id，**不要在同一个 vault 同时安装**生产版和测试版——会互相覆盖。
